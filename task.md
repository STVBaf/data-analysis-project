**《毕业后的第⼀座城市：⻘年城市⽣存⼒指数构建与可视化分析》**

**⼀、项⽬主题**

**主题：** 毕业后的第⼀座城市：⻘年城市⽣存⼒指数构建与可视化分析

**⽬标：** 分析不同城市对⻘年毕业⽣的综合友好度，包括就业机会、租房压⼒、通勤成本、⽣活便利和城市成⻓潜⼒。最终构建**⻘年城市⽣存⼒指数 (YCSI)**，为毕业⽣提供城市选择参考，并通过可视化展

⽰。

# 创新性与亮点：

* 跨领域整合宏观经济、招聘、租房、⽣活便利度、POI 数据
* 构建综合指数体系（YCSI），不仅单指标排名
* 可交互可视化：热⼒地图、雷达图、⽓泡图、四象限分析、聚类分析
* 可提出政策建议或⽣活策略，贴近现实和学⽣关注

**⼆、研究问题**

1. 各城市就业机会与岗位类型分布情况
2. 各城市租房压⼒与租⾦收⼊⽐
3. 通勤便利度：核⼼就业区与低租⾦区的通勤关系
4. 城市⽣活便利度：餐饮、医院、商场、公园、图书馆、地铁等
5. 综合评价：构建⻘年城市⽣存⼒指数（YCSI）
6. 城市聚类与分类，推荐不同类型⻘年适合的城市

**三、研究对象**

15 个典型城市：

北京、上海、⼴州、深圳、杭州、南京、苏州、成都、重庆、武汉、西安、⻓沙、郑州、天津、⻘岛

**六、分析⽅法**

# 描述性分析

* + 柱状图、折线图、箱线图描述城市薪资、租⾦、岗位分布
  + 热⼒图和轮播地图显⽰各城市指标空间分布

# 变量可视化

* + 相关性矩阵、⽹络图：薪资―租⾦―⽣活便利―就业机会
  + 性别/年龄差异分析可选（如果岗位/房租数据⽀持）

# 指数构建

* + 标准化各指标（0-1）
  + 按权重计算 YCSI 指数：

🎼 YCSI = 机会指数\*0.3 + ⽣活便利指数\*0.2 + 成⻓友好指数*0.2*

*- 居住*压*⼒指数*0.2 - 通勤压⼒指数\*0.1

# 聚类分析

* + K-means 或层次聚类，将城市划分为 4 类（⾼机会⾼压⼒、性价⽐型、舒适型、潜⼒型）

1. **预测/模拟**（可选）
   * 根据宏观指标预测未来租⾦/就业变化
   * 蒙特卡洛模拟城市指数可能变化
2. **⽂本分析**（可选）
   * 对社交媒体城市评论做 TF-IDF、词云、共现分析
   * 增强分析趣味性和现实感

1. **⾼级可视化升级点**

# 多维叠加

* + 在热⼒图上叠加城市产业结构的第⼆/第三产业占⽐。
  + Tooltip 显⽰⼤学数量、GDP、⼈均收⼊、metro\_lines 等。

# 时间动态展⽰

* + 所有指标⽀持滑块年份切换，展⽰趋势变化。
  + 四象限和⽓泡图可⽤动画展⽰城市移动轨迹，显⽰压⼒变化和机会增⻓。

# 交互式分析

* + Plotly/Dash/pyecharts 制作交互式地图，点击城市显⽰详细指标表格和折线趋势。
  + 可选筛选维度：产业结构占⽐、⼈⼝规模、⼤学数量。

# 叙事性可视化

* + 增加故事叙事框架：从“毕业⽣选择城市”问题引⼊ → 展⽰城市⽣存⼒指数 → 对⽐典型城市

→ 指出性价⽐城市 → 深度通勤-租房案例分析。

* + ⽤颜⾊、箭头、⼤⼩变化表达逻辑关系，符合教学⼤纲中“可视化叙事能⼒”要求 。

# 多类型编码

* + ⽓泡图：⽓泡⼤⼩=岗位数量，⽓泡颜⾊=⾏业类型，形状区分⼀线/新⼀线城市。
  + 雷达图：使⽤渐变填充区分不同年份数据，增强趋势感。
  + 通勤地图：不同路线颜⾊显⽰不同通勤耗时。

# 可交互导出

* + ⽤⼾可导出每个城市的 CSV 或图表数据，⽅便进⼀步分析。
  + ⽀持点击城市显⽰各指标详细折线和历史变化趋势。
